== Installation ==
1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "job-portal.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (job-portal.zip) that you have downloaded.
 = Upload the entire folder (job-portal) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/job-portal/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "Job Portal" to activate the theme.

5. Complete all of the required inputs in the Appearance => Custimize option and click "Save & Publish" to make the changes LIVE.
